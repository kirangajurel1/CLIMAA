📌 Overview

CLIMAA is a weather dashboard that provides real-time weather and air quality data in a simple and responsive interface.
It allows users to check temperature, humidity, forecasts, and air quality conditions based on location.

The project focuses on clean UI design, modular React architecture, and efficient state management.

✨ Features
🌡️ Real-time weather updates
🌍 Search weather by city
🍃 Air Quality Index (AQI) monitoring
📅 Daily weather forecast
⚙️ User preferences (units & settings)
📱 Fully responsive design
⚡ Fast performance with Vite


🛠️ Tech Stack
Technology	Purpose

Vite	Build tool
Tailwind CSS	Styling

⚙️ Getting Started
1. Clone the repo

cd CLIMAA
2. Install dependencies
npm install
3. Run project
npm run dev
